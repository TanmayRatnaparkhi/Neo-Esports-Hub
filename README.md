# NEO Esports Hub 

A full-stack web platform for hosting, managing, and participating in online esports tournaments. NEO Esports Hub is designed to provide a smooth experience for players, organizers, and viewers with a modern UI and scalable backend architecture.

---

## Features

### User Side

* User registration & authentication
* Browse upcoming and ongoing tournaments
* Tournament registration
* Team creation and management
* Match schedules & brackets
* Player profiles
* Responsive UI for mobile and desktop

### Admin / Organizer Side

* Create and manage tournaments
* Manage teams and players
* Schedule matches
* Update results and standings
* Tournament dashboard

###  Platform

* Secure authentication
* Real-time data handling
* Scalable backend structure
* Clean and modern UI

##  Installation & Setup

### 1ï¸ Clone the repository

```bash
git clone https://github.com/tanmayratnaparkhi/Neo-Esports-Hub.git
cd Neo-Esports-Hub
```

### 2ï¸ Install backend dependencies

```bash
cd backend
npm install
```

### 3ï¸ Setup Firebase

* Create a Firebase project
* Enable Authentication
* Create Firestore database
* Copy credentials into your config file

### 4ï¸ Run backend server

```bash
npm start
```

### 5ï¸ Run frontend

Open `frontend/index.html` in browser
or use Live Server extension in VS Code
